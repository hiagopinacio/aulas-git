#criar repositório
1- abrir git bash
1- ir para a pasta do projeto (com os cd)
$ git init

#commitar
$ git status #verifica se tem mudança no projeto para commitar
$ git add arquivo.txt #adiciona arquivo especifico para commitar
$ git add *.txt #adiciona todos os arquivos .txt para commitar
$ git add . #adiciona todos os arquivos para commitar
""" Após add arquivos:"""
	$ git status #mostra arquivos prontos para serem commitados
	$ git commit -m "mensagem para identificação" #commita e mostra o resultado
	'''Após commitar'''
		$ git status #mostra se não há mais nada para commitar
		
#ignorando arquivos dentro do projeto
1-criar um arquivo chamado .gitignore
2-inserir neste arquivo todos os arquivos a serem ignorados (1 por linha)

#commitando direto:
$ git commit -a -m "mensagem de identificação"
